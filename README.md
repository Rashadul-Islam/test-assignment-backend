## HOW TO INSTALL
  
   ### First create a env file with following properties
     NODE_ENV=
     PORT=
     DATABASE_URL=

     BCRYPT_SALT_ROUNDS=
     JWT_SECRET=
     JWT_REFRESH_SECRET=
     JWT_EXPIRES_IN=
     JWT_ALGORITHM=
     JWT_REFRESH_EXPIRES_IN=

### After that run the following command
  
   - yarn install
   - yarn start
